# Toolsbox for Xiaodeng

## 环境依赖

1. Python v3.9+
2. Pygame v2.0.1+
3. easygui v0.89.2+
4. requests v2.26.0+
5. win10toast

## 部署步骤

1. 安装 Python 3.9 或更高版本

2. 安装必要库
```
$ pip install pygame
$ pip install easygui
$ pip install requests
$ pip install win10toast
```

3. 运行程序
```
$ python3 ToolsBox.py
```
